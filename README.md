## ğŸš€ Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn
- OpenRouter API key

### Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/yourusername/placely.ai.git
   cd placely.ai
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Set up environment variables**

   Create a `.env` file in the root directory:

   ```env
   VITE_OPENROUTER_API_KEY=your_openrouter_api_key_here
   ```

   Get your API key from [OpenRouter](https://openrouter.ai/)

4. **Start the development server**

   ```bash
   npm run dev
   ```

5. **Open your browser**

   Navigate to `http://localhost:5173`

## ğŸ“– Usage Guide

### Setting Your Target

1. Navigate to the **Dashboard**
2. Enter your **Target Company** (e.g., Google, Microsoft)
3. Enter your **Target Role** (e.g., Software Engineer, Data Scientist)
4. Click **Update Dashboard**
5. Your personalized content will be generated!

### Generating a Prep Plan

1. Set your target first (see above)
2. Enter the **Duration** (in weeks)
3. Click **Generate Plan**
4. Review your weekly breakdown of tasks

### Taking an MCQ Quiz

1. Ensure your target is set
2. Navigate to **MCQ Quiz**
3. Select a topic from the available options
4. Answer 5 questions
5. View your score and performance

### Analyzing Your Resume

1. Go to **Resume Feedback**
2. Upload your resume (PDF only, max 5MB)
3. Paste the job description you're targeting
4. Click **Analyze My Resume**
5. Review detailed feedback and recommendations

### Practicing Coding

1. Set your target on the Dashboard
2. Navigate to **Practice Arena**
3. Click on any question to practice on LeetCode
4. Or use the **Code Playground** for custom problems
